# backend

ZENERA is a comprehensive web platform for managing and organizing work schedules within companies.

## 🚀 Features
- Express + TypeScript
- Modular structure (controller, service, routes, dto, validation)
- Database: mongodb
- Optional: Docker, Swagger, Redis, Winston, Husky
- Zod validation
- JWT authentication
- Full testing & linting setup

## 📦 Scripts
- `npm run dev`: Start dev server
- `npm run build`: Build for production
- `npm run start`: Run compiled app
- `npm run lint`: Lint code
- `npm run format`: Format with Prettier
- `npm run test`: Run tests
- `npm run db:seed`: Seed database
- `npm run db:reset`: Reset database

## 🛠️ Environment
See `.env` file.

## 📎 Endpoints
- GET / - Health check
- GET /api/users - List users
- POST /api/users - Create user
- POST /api/auth/login - Login
- Swagger: /docs
